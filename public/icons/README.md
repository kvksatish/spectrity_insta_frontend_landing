# PWA Icons

This directory should contain the following icon sizes for PWA support:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## How to Generate Icons

You can use tools like:
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/
- Or any image editing software to resize your logo

## Requirements
- Use PNG format
- Icons should be square (same width and height)
- Use your app logo or brand identity
- Recommended: Use a simple, recognizable design that works at small sizes
